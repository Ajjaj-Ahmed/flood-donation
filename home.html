What is the DOM in JavaScript?

The DOM (Document Object Model) is a programming interface that allows JavaScript to interact with HTML and XML documents as a structured tree of objects.

How do you select an element by its ID in the DOM?

Use document.getElementById('id') to select an element by its ID.

What method can you use to change the text content of an element?

You can change the text content of an element by setting the textContent property, like element.textContent = 'New Text';

How do you add a new element to the DOM?

You can create a new element using document.createElement('tagName'), then append it using parentElement.appendChild(newElement).
